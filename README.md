
# Real-Time Wild Animal Intrusion Detection System

## How to Run
1. Install Python 3.9+
2. Install dependencies:
   pip install -r requirements.txt
3. Run the system:
   python main.py

## Features
- Real-time camera monitoring
- Detects elephants and wild boars
- Email alert with cooldown logic
- CPU-friendly YOLOv8 Nano model
